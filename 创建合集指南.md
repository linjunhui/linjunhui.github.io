# 如何创建合集（系列文章）

本指南介绍如何在 Jekyll 网站中创建和管理合集（系列文章），例如 "SGLang 源码学习" 系列。

## 快速开始

### 1. 创建系列文章

在 `_posts/` 目录下创建文章，使用以下 front matter 格式：

```yaml
---
title: "SGLang 源码学习 0 - SGLang 架构分析"
date: 2024-01-15
permalink: /posts/2024/01/sglang-architecture-analysis/
categories:
  - SGLang 源码学习
tags:
  - SGLang
  - 源码分析
  - 架构
series: "SGLang 源码学习"  # 系列名称（必须）
series_order: 0  # 系列中的顺序号（必须，数字）
excerpt: "本文是 SGLang 源码学习系列的第一篇文章..."
---
```

### 2. 关键字段说明

- **`series`**: 系列名称，所有同系列文章必须使用相同的名称
- **`series_order`**: 系列中的顺序号，用于排序（0, 1, 2, ...）
- **`categories`**: 分类，建议将系列名称作为分类之一
- **`tags`**: 标签，用于更细粒度的分类

## 文件命名规范

建议使用以下命名格式：

```
YYYY-MM-DD-series-name-part-number.md
```

示例：
- `2024-01-15-sglang-architecture-analysis.md` (第 0 篇)
- `2024-01-20-sglang-tokenizer-analysis.md` (第 1 篇)
- `2024-01-25-sglang-engine-analysis.md` (第 2 篇)

## 系列导航功能

系列导航组件会自动：

1. **显示系列列表**: 在文章顶部显示整个系列的文章列表
2. **高亮当前文章**: 当前文章会以粗体显示，并标注"（当前文章）"
3. **上一篇/下一篇导航**: 自动显示上一篇和下一篇文章的链接
4. **系列统计**: 显示系列总文章数

## 查看系列文章

### 方法 1: 通过分类页面

访问分类归档页面：
```
https://your-site.github.io/categories/#sglang-源码学习
```

### 方法 2: 通过标签页面

访问标签归档页面：
```
https://your-site.github.io/tags/#sglang
```

### 方法 3: 在文章页面

每篇系列文章顶部都会显示系列导航，可以快速跳转到其他文章。

## 最佳实践

### 1. 命名规范

- **系列名称**: 使用清晰、描述性的名称，如 "SGLang 源码学习"
- **文章标题**: 建议包含系列名称和序号，如 "SGLang 源码学习 0 - SGLang 架构分析"
- **文件名**: 使用小写字母和连字符，避免空格和特殊字符

### 2. 顺序管理

- 使用数字作为 `series_order`（0, 1, 2, ...）
- 建议从 0 开始，便于后续插入文章
- 如果需要在中间插入文章，可以使用小数，如 0.5, 1.5

### 3. 分类和标签

- **分类**: 将系列名称作为分类，便于归档
- **标签**: 使用更具体的标签，如 "架构"、"Tokenizer"、"优化" 等

### 4. 内容组织

- 在每篇文章开头说明这是系列的第几篇
- 在文章结尾提供下一篇的预告或链接
- 保持文章长度和深度的一致性

## 高级用法

### 插入中间文章

如果需要在一系列文章中间插入新文章，可以使用小数作为 `series_order`：

```yaml
series_order: 0.5  # 插入在第 0 和第 1 篇之间
```

### 多个系列

一篇文章可以属于多个系列（虽然不常见），只需设置多个 `series` 字段：

```yaml
series:
  - "SGLang 源码学习"
  - "LLM 框架分析"
```

### 自定义系列导航样式

如果需要自定义系列导航的样式，可以编辑 `_includes/series-navigation.html` 文件。

## 常见问题

### Q: 如何修改系列导航的显示位置？

A: 编辑 `_layouts/single.html`，找到 `{% include series-navigation.html %}` 的位置，可以将其移动到文章开头或结尾。

### Q: 系列文章可以不按日期排序吗？

A: 可以。系列导航使用 `series_order` 字段排序，与文章日期无关。

### Q: 如何隐藏某篇文章的系列导航？

A: 不设置 `series` 字段即可，或者设置 `series: false`。

### Q: 系列文章可以跨年份吗？

A: 可以。系列文章可以分布在不同的日期，只要 `series` 名称相同即可。
